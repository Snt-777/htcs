O que seria o DOM?
O DOM é uma sigla para Document Object Model, ele basicamente serve para navegar entre os documentos do site, para entendermos melhor do que se fala, separei uma árvore DOM, essa árvore ilustra a composição de um site:

A relação entre partes na árvore DOM é hierarquia:
document → html → body → div ...
Ou seja o document é 'parent' de html, e o html é 'child' de document, assim como html é 'parent' de body e assim por diante.
