Condições

Temos três tipos de condicões:

Switch case:


 var date = new Date()
 var dia = date.getDay()
 var txt = document.getElementById('sem')

 switch(dia){
     case 1:
         txt.innerText = 'Segunda-Feira'
         break
     case 2:
         txt.innerText = 'Terça-Feira'
         break
     case 3:
         txt.innerText = 'Quarta-Feira'
         break
     case 4:
         txt.innerText = 'Quinta-Feira'
         break
     case 5:
         txt.innerText = 'Sexta-feira'
         break
     case 6:
         txt.innerText = 'Sábado'
         break
     default:
         txt.innerText = 'Domingo'
         break
 }

O break é extremamente importante para não dar erro.
O switch é geralmente utilizado para valores fixos, você pode utilizar ele para inclusive mostrar o dia da semana:

If & Else:


 var idade = 19

 if(idade >= 18){
    console.log('maior de idade')
 }else{
    console.log('menor de idade')
 }

Se quiser tem como utilizar apenas o IF sem utilizar o ELSE, assim irá ocorrer ou não uma ação sem um consequência caso não aconteça.

Else if:


 var idade1 = document.getElementById('idade')//resgata o valor string
 var idade = Number(idade1.value)//transforma o valor em um Number
 var resposta = document.getElementById('res')

 if(idade <= 0 || idade >= 125){
    resposta.innerText = 'Digite uma idade válida'
 }else{
    if(idade < 16){
        resposta.innerText = 'Voto negado!'
    }else if(idade >= 16 && idade < 18 || idade >= 70){
        resposta.innerText = 'Voto opcional!'
    }else{
        resposta.innerText = 'Voto obrigatório!'
    }
 }

As declarações de variáveis são apenas informação, mas o que eu gostaria de demonstrar seria os if, como você pode ver primeiramente tem if dentro do else, basicamente serve para verificar se a idade é válida, esses if dentro do else,se você reparar tem alguns else if que servem como um if extra.
