Temos três tipos de condicões:

Switch case:

O break é extremamente importante para não dar erro.
O switch é geralmente utilizado para valores fixos, você pode utilizar ele para inclusive mostrar o dia da semana:

If & Else:


 var idade = 19

 if(idade >= 18){
    console.log('maior de idade')
 }else{
    console.log('menor de idade')
 }

Se quiser tem como utilizar apenas o IF sem utilizar o ELSE, assim irá ocorrer ou não uma ação sem um consequência caso não aconteça.
