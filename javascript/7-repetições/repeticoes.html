While:

O while se lê como "enquanto", ou seja enquanto algo for verdadeiro, ele irá fazer um loop, quando for falso, irá parar.


 var c = 1 // contador
 

 while(c <= 6){
    console.log('biscoito')
    c++
 }

Bem simples, neste caso nós temos a condição antes de fazer o loop, ou seja, se for verdadeiro → faça, se não → o loop não é feito, ele pula.

do while:

Caso você queira que a condição venha depois, basta utilizar o do while:


 var c = 1 // contador  

 do{
     console.log('biscoito')
     c++
 }while(c <= 6)

A diferença do do while para o while é basicamente que a condição vem depois, ou seja, ele irá fazer o loop → chegou no final → é verdade? → repete o loop, se não apenas continua.

for:



        A        B     C
        ↓        ↓     ↓
 for(var c=1; c <= 4; c++){
     console.log(`${c}`)
 }
  1. var c=1 → Essa parte é a inicial, veja esse c como um contador, nesse caso começa no 1
  2. c <= 4 → Essa é a parte da condição, veja isso como onde deve terminar de repetir
  3. c++ → Essa é a parte do incremento, veja como o passo a ser seguido até o final, ou seja, de 1 em 1 ou de 2 em 2, assim por diante.
