Colocando alertas no site

Aqui eu irei mostrar três tipos de alertas que você pode colocar no seu site, porém lembrando, eles ainda não foram definidos como variáveis, logo qualquer valor que você colocar neles durante o funcionamente não será guardado, só estou mostrando como utilizar eles.

Importante: se você está vendo como colocar um alerta em um site provávelmente você não sabe os métodos de como colocar um código js para rodar em uma página então recomendo acessar Como utilizar scripts de javascript internos e externos

Alerta simples:

  1. window → Essa é a janela do site
  2. alert → Alerta
  3. ('mensagem') → Esse vai ser o lugar onde se coloca o que vai aparecer como alerta

Alertas de confirmação:

  1. window → janela do site
  2. confirm → tipo de alerta de confirmação
  3. ('mensagem') → Esse vai ser o lugar onde se coloca o que vai aparecer como alerta
