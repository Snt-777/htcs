Operadores

Operadores aritiméticos:

  • + → Soma
  • - → Subtração
  • * → Multiplicação
  • / → Divisão
  • % → Resto da divisão
  • ** → Potência
  • () → Parênteses, basicamente serve para indicar o que resolver primeiro

Ordem de precedência:

  1. ()
  2. **
  3. * / %
  4. + -

obs:
adicionar 1 → x ++
remover 1 → x --

Operadores relacionais:

  • > → Maior que
  • < → Menor que
  • >= → Maior que ou igual
  • <= → Menor que ou igual
  • == → igual
  • != → diferente

Operador de identidade:

  1. 10 → Number
  2. '10' → String
  3. == → Igual
  4. === → Operador de identidade

A diferença entre os dois é bem simples, basicamente o == irá verificar como você pode ver na imagem o valor, então mesmo que seja de tipos diferentes caso o valor for o mesmo irá retornar true. Agora o === irá verificar se literalmente tudo é igual desde valor a tipo recebido, por isso ali retorna false, porque o valor é igual, mas o tipo é diferente.

Operadores lógicos:

  • ! → negação
  • && → conjunção
  • || → disjunção

Ordem de procedência dos operadores:

  1. Operadores aritiméticos
  2. Operadores relacionais
  3. Operadores lógicos:
    1. Negação
    2. Conjunção
    3. Disjunção
