Arrays

Ao contrário do que muitos dizem, o array não é tão complicado assim, observe o array como se fosse uma simples lista, e nessa lista o primeiro valor é 0 e assim por diante, mas o array tem alguns comandos próprios que precisam ser mencionados.



  code:
        
  let num = [4, 7, 8, 9]
        
    for(let c in num){
        console.log(num[c])
    }
        
  output:
        
    4
    7
    8
    9

Vou começar pelo início, o let está declarando uma variável e essa variável é o nosso array também conhecido como vetor, o array é declarado com um par de colchetes, dentro deles está nossos elementos e cada elemento tem seu índice, o índice identifica o elemento, o índice começa do 0 e vai até o fim do array, como você pode ver esse array tem 4 elementos sendo eles → 4, 7, 8, 9 ← logo o índice será → 0, 1, 2, 3 ← sendo: 4 = 0 ; 7 = 1 ; 8 = 2 ; 9 = 3.
Logo em seguida temos esse for, mas se você prestar atenção esse for é diferente do for mostrado na parte de repetições, isso porque esse for é voltado para arrays, então ele basicamente vai contar a variavel c que é equivalente a 0 até o fim do array utilizando o índice como base, então o console.log(num[c]) o c está especificando qual elemento ele vai querer do num.
É bem simples a coisa que dificulta são os loops

Comandos utilizados no array:

  • num.push(5) → irá adicionar ao final do array um elemento, neste caso o número 5
  • num.length → irá contar a quantidade de elementos dentro do array(esse comando não utiliza parênteses)
  • num.sort() → irá ordenar os valores de formar crescente dentro do array, mas caso um número seja adicionado depois, ele não irá entrar organizado.
  • num.indexOf(8) → irá procurar o elemento dentro do array, caso tenha irá retornar seu índice, caso não exista o elemento irá retornar o valor -1
  • list.splice(0, 1) → irá remover o item da lista, nesse caso o 0 significa o índice do elemento a ser removido e o 1 significa a quantidade elementos que serão removidos depois desse índice que serão removidos, por exemplo se eu colocase no lugar de 1 → 2 basicamente iria remover o elemento do índice 0 e o elemento do índice 1, mas como o número é 1 irá remover apenas o elemento do índice 0
    • list.splice(num.indexOf(0), 1) → basicamente nesse método basta colocar o elemento utilizando o index que você pode remove-lo sem saber o índice.
