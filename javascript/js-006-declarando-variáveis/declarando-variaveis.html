Comandos fundamentais

Irei mostrar como utilizar alguns comandos básicos e úteis, também irei mostrar mensagem como declarar variáveis.

Declarando variáveis:

Declarar uma variável é uma coisa bem simples.

  1. var → Comando de declaração de variável
  2. nome → Nome da variável
  3. 'javascript' → String. Conteúdo dentro da variável
  4. 15 → Number. Conteúdo dentro da variável

obs (Essas são variáveis simples, básicas, sem especifícações)

Regras de nome para as variáveis:

  • Podem começar com & ou _
  • Não podem começar com números
  • É possível usar letras ou números
  • É possível usar acentos e símbolos
  • Não podem conter espaços
  • Não podem ser palavras reservadas

Comandos básicos:

Vou fazer uma lista com os comandos fundamentais, mas não acho necessário utilizar imagens.

  • // comentário → Comentário de única linha
  • /* comentário */ → Comentário de mais de uma linha
  • Number(X) / String(X) → Converter variável para tipos diferentes
  • Number.parseFloat(X) / Number.parseInt(X) → Converção expecífica
  • `Seja bem vindo ao meu site feito em ${nome}` → Formatação String ,necessita o uso de crase
  • s.toLowerCase() → Letras minúsculas
  • s.toUpperCase() → Letras maiúsculas
  • s.length → quantidade de letras na String
  • n.toFixed(x) → números depois da vírgula
  • n.toLocaleString('pt-BR', {style: 'currency', currency: 'BRL'}) → vai mostrar a variável n em formato monetário
