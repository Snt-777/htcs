Âncoras

Como utilizar âncoras externas e internas

Âncoras são bem simples de serem usadas porém tem alguns detalhes que temos que observar sobre elas, eu irei explicar esse detalhes logo a seguir.

Âncoras externas:

Olá sou o ian, logo abaixo estarei disponibilizando meu perfil no github, segue lá!

perfil github
  1. <a> → Essa é a tag de âncora, o link que irá aperecer na tela vem antes do fechamento da tag
  2. href="..." → Esse parâmetro é a referência do link, ou seja o que estiver dentro dele vai ser para onde o link irá levar a gente, nesse caso seriamos redirecionados para o meu perfil no git hub. Para pegar um link externo é bem simples basta acessar a página que você deseja colocar como link e copiar a URL da página
  3. hreflang="..." → Essa é a lingua do site que você está redirecionando o usuário, ela irá ajudar o mecanismo de busca a saber se é para traduzir ou não.
  4. target="..." → Esse é bem importante, o parâmetro target irá definir como será aberto o link, no caso está blank porque como é uma página fora do meu site, ele irá abrir em uma nova guia e não irá fechar meu site, assim se o usuário quiser voltar para o meu site ainda estará aberto lá (vou dar mais detalhes sobre os targets na parte: os tipos de target)
  5. rel="..." → Esse é o parâmetro rel, ele irá ajudar o mecanismo de busca a saber se o link irá para uma página externa ou interna do seu site, neste caso é externa logo o valor de rel será external

Âncoras internas:

Vá para página de listas clicando aqui

  1. ../ → Esse sinal serve para volta 1 arquivo(pasta), serve para selecionar itens de pastas anteriores
  2. target="_self" → Neste caso não é necessário utilizar o _self, mas eu utilizei para explicar apenas, o _self serve para abrir uma página sobre a já aberta, essa configuração é padrão, não precisa por ela.
  3. rel="next" → O rel no caso de páginas internas tem dois tipos, e eles servem para ajudar o navegador:
    • next → próxima página no site
    • prev → página anterior no site
