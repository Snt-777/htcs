Como utilizar âncoras para download

Para utilizar o download com âncoras é basicamente a mesma coisa de usar elas para acessar sites, mas tem algumas modificações que devemos levar em conta.

download:

Mensagem motivacional, aperte no link abaixo para baixar a mensagem motivacional:

download
  1. href="(documento download)" → Eu já expliquei href, mas neste caso nós não colocamos um link de um site ou página, mas sim o arquivo que queremos que seja baixado(neste caso utilizei o arquivo zip porque apenas de apertar já baixa, dependendo do arquivo não baixa com o click).
  2. download="..." → No parâmetro download colocamos exatamente o mesmo nome de arquivo para indicar se será baixado
  3. type="..." → Ai que começa as variações, no type nós colocamos o tipo de documento que será baixado. Aqui abaixo vai uma lista com os principais:
    • application/zip
    • application/pdf
    • text/html
    • text/css
    • text/javascript
    • video/mp4
    • video/H264
    • video/JPEG
    • audio/aac
    • audio/mpeg
    • font/ttf
    • image/jpeg
    • image/png

Existe bem mais types, eu coloquei apenas os principais, se caso algum dia você quiser saber ou precisar aperte aqui para ir a um site que mostra todos os media types.
