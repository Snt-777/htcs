Como utilizar listas

Nós temos dois tipos de listas, as listas ordenadas e as desordenadas.

Lista ordenada:

Meus jogos favoritos:

  1. Rocket league
  2. Destiny 2
  3. Roblox
  4. Mario
  5. Sonic
  6. Cs go
  1. <p> → Neste caso eu estou utilizando a tag <p> para apenas falar sobre o significado da lista, basicamente dar uma explicação ao usuário
  2. <ol> → Essa tag é a lista ordenada, dentro dela é onde você coloca os itens.
  3. type="..." → Este é um parâmetro utilizado nas listas em geral, ele serve no caso da lista ordenada para indicar de qual forma ela será ordenada.
    Dentre elas temos:
    • 1 → Este já esta indicado por padrão nas listas ordenadas, mas serve para mostrar ordenadamente em números
    • A → Este serve para mostrar ordenado em letras maiúsculas
    • a → Este serve para mostrar ordenado em letras minúsculas
    • I → Este serve para mostrar ordenado em números romanos maiúsculos
    • i → Este serve para mostrar ordenado em números romanos minúsculos
  4. <li> → É utilizado para indicar um item da lista.

(Obs: Bom não sei se você percebeu, mas utilizei lista desordenada e utilizei lista dentro de lista na lista acima.)

Lista desordenada:

Atividades perfeitas para dias chuvosos

  • Desenvolver sites
  • Ler um livro
  • Criar uma história
  • Jogar
  • Estudar algo que goste
  1. <ul> → tag de lista desordenada
  2. type="..." → Este é um parâmetro das listas en geral, no caso da lista desordenada, ele serve apenas para modificar o ícone que aparece no lado.
    Nele temos:
    • none → Para não aparecer ícone
    • disc → Para aparecer um circulo fechado
    • circle → Para aparecer um circulo com um buraco no meio
    • square → Para aparecer um quadrado
  3. <li> → como eu disse anteriormente, serve para indicar um item da lista.

  1. <dl> → tag que irá determinar a lista.

Mesclando listas:

Isto é uma coisa bem simples, basta utilizar uma lista dentro da outra, porém isso pode parecer confuso as vezes, então neste caso irei mostrar como mesclar.

O pódio:

  1. Marcelo
    • pontos: 99,3
  2. Júlia
    • pontos: 99,1
  3. Jorge
    • pontos: 98,8
  1. <p> → Novamente utilizando esse parágrafo para mostrar ao usuário o significado da lista
  2. <ol> → Aqui podemos ver a abertura da lista ordenada
  3. <li> → Isso é um item da lista ordenada aberta logo acima
  4. <ul> → Está é a abertura da lista desordenada
  5. <li> → Aqui podemos ver um item da lista desordenada

Explicação:

Eu sei que ficou bem confuso por isso irei explicar. Vou começar do início e ir descendo, a tag <p> está sendo apenas usada para falar ao usuário o significado da lista, logo abaixo temos a abertura da lista com a tag <ol> (se você conferir essa tag de abertura inicial da lista ordenada só se fecha lá no final) logo abaixo da tag inicial da lista temos um item e abaixo deste item um lista desordenada a tag <ul> essa lista desordenada tem um item dentro dela e logo após ela fecha, e vem denovo um item da ordenada, abre desordenada, item da desordenada, e vai assim por diante, porém a única coisa que você tem que ter em mente é que a lista desordenada está dentro da lista ordenada, ou seja pela regra da hierarquia, a lista desordenada pertence a ordenada logo o item que está dentro da desordenada está seguindo essa hierarquia também, ordenada > desordenada > item, por isso funciona.
