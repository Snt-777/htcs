Como colocar diferenças no texto

Vou passar aqui algumas tags utilizadas para modificar o texto, bem simples de colocar mas como são muitas acaba que esquecemos como utilizar. Temos algumas tags não semânticas, porém não vou passar elas com explicações porque usamos em geral as tags semânticas.

  • Negrito
    • <b> → negrito não semântico
    • <strong> → negrito da forma semântica (texto em destaque)
  • Itálico
    • <i> → itálico não semântico
    • <em> → itálico da forma semântica

Mark:

O mark eu irei explicar a parte, porque ele é uma tag simples porém com umas observações.

texto
  1. <mark> → tag mark(texto importânte)
  2. style="..." → parâmetro style(css) onde será você irá passar o estilo que você vai querer, neste caso eu usei o style na tag, mas você pode utilizar no head para efeitos gerais.
  3. texto → a mensagem que será marcada

código-fonte/pré-formatação:

Esses dois vou ter que dar uma breve explicada, o código fonte que é a tag <code> nada mais é um espaçamento, ou seja ele deixa as palavras com a mesma espessura para que fique mais fácil de ler o "código".
Já a pré-formatação que é a tag <pre> basicamente é para que o que o desenvolvedor digitou no código da página fique ordenado da forma que ele digitou ou seja contar os espaços quebras de linhas e etc... Isso é geralmente utilizado para escrever trechos de códigos em uma página web. 

        
            n1 = input('digite sua nota:')
            if n1 > 7:
                print('aprovado!')
            else:
                print('reprovado!')
  1. <pre> → tag de pré-formatação
  2. <code> → tag de espaçamento
  3. código → o código em si escrito da mesma forma que apareceu na tela

usando isso tudo junto se faz o resultado logo acima da imagem, se você não quiser esses espaços da borda da tela basta colocar tudo coladinha na borda da linha.
